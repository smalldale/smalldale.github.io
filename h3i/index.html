CW Error Finder

- unofficial -

Cara Penggunaan

  1. Siapkan response file csv dari Voucher Unlock Non HI7 Status
  2. Klik "Choose csv file..." lalu pilih file report (response file) csv yang telah disiapkan
  3. Hasil pencarian akan ditampilkan apabila error ditemukan dalam file csv
*Kode error yang ditemukan berdasarkan adanya kata "Your request cannot be processed at this time, please try again later."
